Exploration of <details> element

Notes

  • the 'open' attribute indicates whether the details will be shown to the user on page load; default = false
  • If the element is omitted the heading "details" will be used. This is the responsibility of the user agent.
  • In these examples, the gray border is being added to the <details> element through local CSS.
